Links and buttons

The elements <a> are reserved for in-line links. These unvisited and visited links are colored blue and underlined to support color blind and low-vision readersGuidelines for visualizing links. There are two other links used outside text: external links and buttons. External links <a class='ext'> are more visible as context doesn't help the reader recognizing them:
external link

References

You a can add a reference to the text, which is a special type of link, by wrapping the link inside a <span class='ref'> tag. The resulting link becomes a reference which is then indicated by a ref superscript right where you want to put the reference.
